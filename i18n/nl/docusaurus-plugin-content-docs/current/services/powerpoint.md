# PowerPoint

PowerPoint is de Microsoft 365-app voor presentaties, workshops en visuele verhalen in een diapresentatie.

## Het beste voor

Gebruik PowerPoint voor vergaderingen, trainingen, verkoopgesprekken, updates voor het management en herbruikbare presentaties.

## Let op

PowerPoint wordt vaak te veel gebruikt als publicatieformaat. Gebruik SharePoint als mensen een stabiele pagina, doorzoekbare inhoud of een intranetbron nodig hebben.
