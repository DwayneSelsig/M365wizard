# SharePoint

SharePoint biedt sites, pagina's, lijsten, bibliotheken, machtigingen en publicatiepatronen voor Microsoft 365-inhoud.

## Het beste voor

Gebruik SharePoint voor intranetinhoud, documentbibliotheken, gepubliceerde informatie, gestructureerde lijsten, teamsites en organisatiebronnen.

## Let op

SharePoint kan rommelig worden wanneer eigenaarschap onduidelijk is. Elke belangrijke site of bibliotheek heeft een eigenaar, doel en levenscyclus nodig.
