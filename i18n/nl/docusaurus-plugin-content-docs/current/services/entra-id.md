# Entra ID

Microsoft Entra ID is het identiteitsplatform achter aanmelden, toegang, groepen en veel beveiligingspatronen in Microsoft 365.

## Het beste voor

Gebruik Entra ID om identiteit, toegang, verificatie, groepen, bedrijfstoepassingen en voorwaardelijke toegang te beheren.

## Let op

Identiteitsinstellingen raken elke dienst. Wijzigingen moeten worden getest, gecommuniceerd en beheerd door mensen die zowel beveiliging als gebruikersimpact begrijpen.
