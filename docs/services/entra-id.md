# Entra ID

Microsoft Entra ID is the identity platform behind sign-in, access, groups, and many Microsoft 365 security patterns.

## Best For

Use Entra ID to manage identity, access, authentication, groups, enterprise applications, and conditional access.

## Be Careful With

Identity settings affect every service. Changes should be tested, communicated, and owned by people who understand both security and user impact.
