# Power BI

Power BI helps create reports, dashboards, and data models for business insight.

## Best For

Use Power BI when people need repeatable reporting, shared dashboards, governed datasets, and interactive analysis.

## Be Careful With

Do not treat every spreadsheet as a Power BI project. Start with the decision people need to make and the data quality available to support it.
