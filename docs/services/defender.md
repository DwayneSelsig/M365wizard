# Defender

Microsoft Defender services help protect identities, devices, email, apps, and cloud resources.

## Best For

Use Defender as part of a broader security operating model: detection, prevention, investigation, and response.

## Be Careful With

Security tools need ownership and process. Alerts, policies, and reports only help when someone is responsible for reviewing and acting on them.
