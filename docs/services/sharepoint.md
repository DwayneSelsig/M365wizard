# SharePoint

SharePoint provides sites, pages, lists, libraries, permissions, and publishing patterns for Microsoft 365 content.

## Best For

Use SharePoint for intranet content, document libraries, published information, structured lists, team sites, and organization resources.

## Be Careful With

SharePoint can become messy when ownership is unclear. Every important site or library needs an owner, a purpose, and a lifecycle.
