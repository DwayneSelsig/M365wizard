# Word

Word is the primary Microsoft 365 app for documents with rich text, structure, review, and formatting.

## Best For

Use Word for reports, policies, proposals, letters, procedures, and documents that need review or formal structure.

## Be Careful With

Do not use old Word files as unofficial templates. If many people need the same starting point, publish an approved template.
