# PowerPoint

PowerPoint is the Microsoft 365 app for presentations, workshops, and visual storytelling in a deck format.

## Best For

Use PowerPoint for meetings, training, sales conversations, executive updates, and reusable presentation decks.

## Be Careful With

PowerPoint is often overused as a publishing format. If people need a stable page, searchable content, or an intranet resource, use SharePoint.
