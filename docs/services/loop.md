# Loop

Loop helps people co-create flexible content and reusable components across Microsoft 365.

## Best For

Use Loop for collaborative notes, evolving ideas, shared agendas, small working spaces, and content that changes while a group thinks together.

## Be Careful With

Loop is not always the right final home for official information. Move stable or published content into SharePoint or another governed location when the work matures.
