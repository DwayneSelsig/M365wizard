# Viva Engage

Viva Engage is a community and employee engagement platform in Microsoft 365.

## Best For

Use Viva Engage for communities of practice, leadership conversations, open questions, employee engagement, and organization-wide discussion.

## Be Careful With

Viva Engage is not the best place to run detailed project work. Use Teams when the group needs files, meetings, tasks, and daily coordination.
