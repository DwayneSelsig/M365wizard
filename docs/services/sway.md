# Sway

Sway helps create lightweight, web-style stories and presentations.

## Best For

Use Sway for visual explainers, newsletters, event recaps, and simple stories that people read on their own.

## Be Careful With

Do not use Sway for official intranet content that needs strong navigation, ownership, governance, and findability. SharePoint is usually better for that.
