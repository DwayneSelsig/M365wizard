# When To Use Organization Assets

Organization assets help people find approved templates, images, and brand materials inside Microsoft 365.

## Quick Answer

Use organization assets when content should be reused widely and consistently, especially Office templates, logos, brand images, and approved document starting points.

## Use This Pattern When

- Many people create similar Word, Excel, or PowerPoint files.
- Brand consistency matters.
- Templates change over time.
- People currently copy old files as a starting point.
- You want approved assets to appear where people already work.

## Do Not Use It For Everything

An organization assets library is not a general dumping ground. It should contain curated, trusted material. If every team can add anything, people will stop trusting it.

## What Needs Governance

Before publishing assets, decide who owns them, who can update them, how reviews happen, and how old templates are retired.

The technical setup is only half the pattern. The value comes from making the approved choice easier than the unofficial workaround.
